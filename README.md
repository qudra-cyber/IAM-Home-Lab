# Identity and Access Management (IAM) Home Lab

## Project Overview

This project demonstrates Identity and Access Management (IAM) concepts using Keycloak.

The lab simulates a company environment with multiple departments and users while implementing Role-Based Access Control (RBAC), Multi-Factor Authentication (MFA), and Joiner-Mover-Leaver (JML) workflows.

## Technologies Used

* Keycloak
* Windows 11
* RBAC
* MFA
* Identity Governance Concepts

## Company Structure

Departments:

* HR
* IT
* Finance
* Sales

## Tasks Completed

### User Management

* Created user accounts
* Managed user credentials
* Updated user information
* Performed access reviews

### Group Management

* Created departmental groups
* Assigned users to departments
* Validated membership assignments

### Role-Based Access Control

Created roles:

* Employee
* Manager
* Admin

Assigned roles to users based on job responsibilities.

### Multi-Factor Authentication

Configured OTP authentication to strengthen account security.

### Joiner-Mover-Leaver Process

#### Joiner

Created a new employee account (Sarah).

#### Mover

Assigned Sarah a Manager role.

#### Leaver

Demonstrated access removal and account lifecycle management procedures.

## Skills Demonstrated

* Identity & Access Management
* User Provisioning
* Role-Based Access Control (RBAC)
* Multi-Factor Authentication (MFA)
* Access Reviews
* Joiner-Mover-Leaver (JML)
* Identity Governance
